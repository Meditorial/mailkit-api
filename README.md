Requirements
------------

Igloonet/MailkitApi requires PHP 7.1 or higher.

Installation
------------

The best way to install Igloonet/MailkitApi is using  [Composer](http://getcomposer.org/):

```sh
$ composer require igloonet/mailkit-api
```
